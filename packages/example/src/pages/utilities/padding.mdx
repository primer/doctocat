Padding utilities are based on a global [spacing scale](/css/support/spacing) which helps keep horizontal and vertical spacing consistent. These utilities help us reduce the amount of custom CSS that could share the same properties, and allows to achieve many different page layouts using the same styles.

## Shorthand

Since padding utilities have many variations and will be used in many places, we use a shorthand naming convention to help keep class names succinct.

| Shorthand | Description              |
| --------- | ------------------------ |
| p         | padding                  |
| t         | top                      |
| r         | right                    |
| b         | bottom                   |
| l         | left                     |
| x         | horizontal, left & right |
| y         | vertical, top & bottom   |
| 0         | 0                        |
| 1         | 4px                      |
| 2         | 8px                      |
| 3         | 16px                     |
| 4         | 24px                     |
| 5         | 32px                     |
| 6         | 40px                     |

_**Note:** The blue in the examples below represents the element, and the green represents the padding._

## Uniform padding

Use uniform spacing utilities to apply equal padding to all sides of an element. These utilities can be used with a spacing scale from 0-6.

```html live
<div>
  <div class="p-0 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-0</div>
  </div>
  <div class="p-1 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-1</div>
  </div>
  <div class="p-2 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-2</div>
  </div>
  <div class="p-3 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-3</div>
  </div>
  <div class="p-4 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-4</div>
  </div>
  <div class="p-5 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-5</div>
  </div>
  <div class="p-6 d-inline-block bg-green-light border mr-3">
    <div class="d-inline-block bg-white">.p-6</div>
  </div>
</div>
```

## Directional padding

Use directional utilities to apply padding to an individual side, or the X and Y axis of an element. Directional utilities can change or override default padding, and can be used with a spacing scale of 0-6.

```html live
<div>
  <div class="pt-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.pt-3</div>
  </div>
  <div class="pr-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.pr-3</div>
  </div>
  <div class="pb-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.pb-3</div>
  </div>
  <div class="pl-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.pl-3</div>
  </div>
  <div class="py-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.py-3</div>
  </div>
  <div class="px-3 bg-green-light d-inline-block border mr-3">
    <div class="d-inline-block bg-white">.px-3</div>
  </div>
</div>
```

## Responsive padding

All padding utilities can be adjusted per [breakpoint](/css/support/breakpoints) using the following formula: <br /> `p-[direction]-[breakpoint]-[spacer]`. Each responsive style is applied to the specified breakpoint and up.

```html live
<div class="px-sm-2 px-lg-4 d-inline-block bg-green-light border">
  <div class="d-inline-block bg-white">
    .px-sm-2 .px-lg-4
  </div>
</div>
```

## Responsive container padding

`.p-responsive` is a padding class that adds padding on the left and right sides of an element. On small screens, it gives the element padding of `$spacer-3`, on mid-sized screens it gives the element padding of `$spacer-6`, and on large screens, it gives the element padding of `$spacer-3`.

It is intended to be used with [container styles](/css/objects/grid#containers)

```html live
<div class="container-lg p-responsive bg-green-light border">
  <div class="bg-white">
    .container-lg .p-responsive
  </div>
</div>
```
